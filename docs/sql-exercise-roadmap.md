# SQL 단계별 연습 문제

이 문서는 [HR 연습 데이터베이스 스키마](./hr-schema.md)를 이용해 SQL을 기초부터 고급까지 순서대로 학습할 수 있도록 구성한 문제 목록입니다.

- 모든 문제는 SQLite 문법을 기준으로 합니다.
- 별도 지시가 없다면 결과 행 순서는 평가하지 않습니다.
- 정렬 조건이 제시된 문제는 행 순서까지 평가합니다.
- 날짜는 `YYYY-MM-DD` 형식의 `TEXT`로 저장되어 있다고 가정합니다.

## 1단계: 기본 조회

학습 목표: `SELECT`, 열 선택, 별칭, 중복 제거

1. `employees` 테이블의 모든 데이터를 조회하세요.
2. 모든 직원의 `first_name`, `last_name`, `email`을 조회하세요.
3. 모든 부서의 `department_id`, `department_name`을 조회하세요.
4. 직원의 `first_name`과 `last_name`을 각각 `firstName`, `lastName`이라는 별칭으로 조회하세요.
5. 직원에게 지정된 `job_id` 목록을 중복 없이 조회하세요.
6. 직원이 소속된 `department_id` 목록을 중복 없이 조회하세요. 소속 부서가 없는 직원의 `NULL`도 결과에 포함합니다.

## 2단계: 조건과 정렬

학습 목표: `WHERE`, 비교 연산자, 논리 연산자, `NULL`, `LIKE`, `IN`, `BETWEEN`, `ORDER BY`

7. 급여가 10,000 이상인 직원의 이름과 급여를 조회하세요.
8. 급여가 5,000 이상 10,000 이하인 직원의 이름과 급여를 조회하세요.
9. `job_id`가 `SW_ENG` 또는 `SA_REP`인 직원을 조회하세요.
10. 커미션을 받는 직원의 이름과 `commission_pct`를 조회하세요.
11. 소속 부서가 없는 직원을 조회하세요.
12. `last_name`이 `S`로 시작하는 직원을 조회하세요.
13. 2020-01-01 이후 입사한 직원을 조회하세요. 해당 날짜도 포함합니다.
14. 급여가 8,000 이상이면서 커미션을 받지 않는 직원을 조회하세요.
15. 모든 직원을 급여가 높은 순서로 조회하고, 급여가 같으면 `employee_id` 오름차순으로 정렬하세요.
16. 도시 이름을 알파벳 오름차순으로 조회하세요.

## 3단계: 표현식과 함수

학습 목표: 문자열·숫자·날짜 표현식, `CASE`, `COALESCE`

17. 직원의 전체 이름을 `first_name`과 `last_name` 사이에 공백을 넣어 `full_name`으로 조회하세요.
18. 직원의 월 급여와 연봉을 함께 조회하세요. 연봉은 월 급여에 12를 곱해 계산합니다.
19. 커미션이 없는 경우 `commission_pct` 대신 `0`을 표시하세요.
20. 직원의 이메일 주소를 소문자로 변환해 조회하세요.
21. 각 직원의 `last_name`과 이름의 글자 수를 조회하세요.
22. 급여가 15,000 이상이면 `HIGH`, 8,000 이상이면 `MEDIUM`, 나머지는 `LOW`로 표시하세요.
23. 직원의 입사 연도만 추출해 조회하세요.
24. 커미션을 포함한 예상 월 보수를 계산하세요. 커미션이 없으면 급여만 표시합니다.

## 4단계: 집계와 그룹화

학습 목표: 집계 함수, `GROUP BY`, `HAVING`, 조건부 집계

25. 전체 직원 수를 조회하세요.
26. 직원 급여의 최솟값, 최댓값, 평균값을 조회하세요.
27. 부서별 직원 수를 조회하세요. 소속 부서가 없는 직원도 하나의 그룹으로 포함합니다.
28. 직무별 평균 급여를 조회하세요.
29. 직원이 5명 이상인 부서의 `department_id`와 직원 수를 조회하세요.
30. 평균 급여가 8,000 이상인 직무의 `job_id`와 평균 급여를 조회하세요.
31. 부서별로 커미션을 받는 직원 수와 받지 않는 직원 수를 각각 조회하세요.
32. 입사 연도별 직원 수를 조회하고 연도 오름차순으로 정렬하세요.

## 5단계: 조인

학습 목표: 내부 조인, 외부 조인, 다중 테이블 조인, 셀프 조인

33. 각 직원의 이름과 소속 부서 이름을 조회하세요.
34. 각 직원의 이름과 직무 이름을 조회하세요.
35. 각 부서의 이름과 부서가 위치한 도시를 조회하세요.
36. 직원의 이름, 부서 이름, 도시를 조회하세요.
37. 직원의 이름, 직무 이름, 국가 이름, 지역 이름을 조회하세요.
38. 모든 부서와 소속 직원 수를 조회하세요. 직원이 없는 부서도 포함합니다.
39. 모든 직원의 이름과 직속 관리자 이름을 조회하세요. 관리자가 없는 직원도 포함합니다.
40. 각 부서의 이름과 부서 책임자 이름을 조회하세요. 책임자가 없는 부서도 포함합니다.
41. 직원이 한 명도 없는 부서를 조회하세요.
42. 근무지가 배정되지 않은 부서를 조회하세요.

## 6단계: 서브쿼리와 집합 연산

학습 목표: 스칼라·다중 행·상관 서브쿼리, `EXISTS`, `UNION`, `INTERSECT`, `EXCEPT`

43. 전체 직원 평균 급여보다 높은 급여를 받는 직원을 조회하세요.
44. 자신의 부서 평균보다 높은 급여를 받는 직원을 조회하세요.
45. 회사에서 가장 높은 급여를 받는 직원을 조회하세요. 공동 1위도 모두 포함합니다.
46. 직무별 최소 급여보다 실제 급여가 낮은 직원을 조회하세요.
47. 한 명 이상의 직원을 보유한 부서만 `EXISTS`를 사용해 조회하세요.
48. 과거 직무 이력이 존재하는 직원을 조회하세요.
49. 현재 또는 과거에 근무한 적이 있는 모든 `job_id`를 중복 없이 조회하세요.
50. 현재는 사용되지 않지만 과거 직무 이력에는 존재하는 `job_id`를 조회하세요.

## 7단계: CTE와 윈도 함수

학습 목표: `WITH`, 순위, 누적 집계, 파티션 집계, 이전·다음 행 비교

51. CTE로 부서별 평균 급여를 계산한 뒤 평균 급여가 가장 높은 부서를 조회하세요.
52. 각 직원의 급여와 전체 평균 급여의 차이를 조회하세요.
53. 부서별로 직원의 급여 순위를 구하세요. 같은 급여에는 같은 순위를 부여합니다.
54. 각 부서에서 급여가 높은 상위 3명의 직원을 조회하세요. 공동 순위도 포함합니다.
55. 직무별 직원 급여와 해당 직무 평균 급여를 한 행에 함께 표시하세요.
56. 입사일 순서에 따라 직원 수의 누적 합계를 조회하세요.
57. 각 직원과 같은 부서에서 바로 이전에 입사한 직원의 이름과 입사일을 조회하세요.
58. 전체 급여 합계에서 각 직원의 급여가 차지하는 비율을 조회하세요.

## 8단계: 고급 분석

학습 목표: 재귀 CTE, 다단계 집계, 관계 분석, 연속 기간 분석

59. 재귀 CTE를 사용해 최고 관리자부터 시작하는 직원 조직도를 계층 깊이와 함께 조회하세요.
60. 각 관리자의 모든 직속 및 간접 부하 직원 수를 조회하세요.
61. 지역별 직원 수, 평균 급여, 최고 급여를 조회하세요. 직원이 없는 지역도 포함합니다.
62. 부서별 급여 총액이 전체 급여 총액에서 차지하는 비율을 조회하고 비율이 높은 순서로 정렬하세요.
63. 각 직무에서 `NTILE(10)` 기준 급여 상위 10%에 해당하는 직원을 조회하세요.
64. 각 직원의 현재 직무와 가장 최근 과거 직무를 한 행에 표시하세요. 직무 이력이 없는 직원도 포함합니다.
65. `job_history`를 이용해 직원별 과거 총 근무 일수를 계산하세요.
66. 직원별 직무 변경 횟수를 구하고 가장 많이 변경한 직원을 조회하세요. 공동 1위도 포함합니다.
67. 같은 부서에서 같은 해에 입사한 직원 쌍을 조회하세요. 동일한 쌍이 반대 순서로 중복되지 않게 합니다.
68. 부서별 급여 중앙값을 계산하세요. 직원 수가 짝수라면 가운데 두 급여의 평균을 사용합니다.
69. 각 국가에서 평균 급여가 가장 높은 부서를 조회하세요. 공동 1위도 포함합니다.
70. 직원이 한 번도 배정되지 않은 직무와 직원이 한 번도 배정되지 않은 부서를 각각 찾아 하나의 결과로 표시하세요.

- 15번: 급여 내림차순과 직원 ID 오름차순
- 16번: 도시 이름 오름차순
- 32번: 입사 연도 오름차순
